## üìù What Each Secret Does

- **JWT_ACCESS_SECRET**: Signs access tokens for user authentication
- **JWT_REFRESH_SECRET**: Signs refresh tokens for token renewal
- **JWT_SECRET**: General JWT signing (backward compatibility)
- **COOKIE_SECRET**: Signs cookies to prevent tampering
- **SESSION_SECRET**: Encrypts session data
- **ENCRYPTION_KEY**: Encrypts sensitive data in database
